Dúvidas frequentes

Encontre respostas rápidas sobre a ClimateWise, funcionalidades da plataforma e uso dos dados climáticos.

help_outline O que é a ClimateWise?

A ClimateWise é uma plataforma de inteligência climática que centraliza dados, analisa padrões e apresenta riscos ambientais de forma visual e intuitiva para apoiar decisões.

cell_tower De onde vêm os dados climáticos?

Os dados podem ser integrados a fontes meteorológicas públicas, estações próprias e sensores, dependendo do cenário de uso definido pelo usuário ou organização.

dashboard_customize É possível personalizar os painéis e indicadores?

Sim. Os dashboards podem ser configurados conforme a necessidade, ajustando indicadores, regiões monitoradas e tipos de alerta exibidos.

notifications_active Como funcionam os alertas inteligentes?

Os alertas são disparados quando os indicadores climáticos atingem limites definidos, como calor extremo, chuvas intensas, ventos fortes ou umidade crítica.

groups Quem pode usar a ClimateWise?

A plataforma pode ser utilizada por empresas, governos, escolas, pesquisadores e qualquer pessoa que precise acompanhar cenários climáticos com mais precisão.

security Meus dados ficam seguros?

Sim. O projeto considera boas práticas de segurança e controle de acesso, com gestão de usuários e permissões em camadas.
